Win10系统下，使用hexo（5.3.0）+ Github 搭建博客
发表于2021-01-09|更新于2021-01-10|教程|blog•教程
Hexo 是基于 Node.js 的静态博客框架，可以部署到任意的 Git 平台（Github、Gitlab、Coding、Gitee…）以及自己的服务器 / VPS。 一、软件安装 1.安装Node.js（推荐LTS版） 下载地址: 这里 2.安装Git 下载地址: 这里 （上面的两个都以默认设置安装就行，如果你需要更改安装路径需要自己配置环境变量） 安装完成后新建一个存放博客的文件夹（不要在桌面，eg:D:\blog），在新建的文件夹中右键打开git bash here 即可打开Git 的Bash终端 在Git Bash分别输入以下命令并回车： git --version #查看Git版本号 NPM -v #查看NPM版本号 3.安装Hexo Git Bash输入命令： npm install -g hexo-cli hexo -v #查看Hexo版本号 二、本地搭建 Hexo的基本命令 hexo clean #清除生成的博客静态文件 hexo g #生成博客静态文件=命令hexo generate hexo s #启动博客的本地预览=命令hexo server hexo d ...
