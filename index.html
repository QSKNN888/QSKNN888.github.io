title: Win10系统下，hexo（5.3.0）+ Github 搭建博客

Hexo 是基于 Node.js 的静态博客框架，可以部署到任意的 Git 平台（Github、Gitlab、Coding、Gitee…）以及自己的服务器 / VPS。

一、软件安装
1.安装Node.js（推荐LTS版）
下载地址: https://nodejs.org/en/download/

2.安装Git
下载地址: https://git-scm.com/

（上面的两个都以默认设置安装就行，如果你需要更改安装路径需要自己配置环境变量）

安装完成后新建一个存放博客的文件夹（不要在桌面，eg:D:\blog），在新建的文件夹中右键打开git bash here 即可打开Git 的Bash终端

在Git Bash分别输入以下命令并回车：
git –version #查看Git版本号
NPM -v #查看NPM版本号

3.安装Hexo
Git Bash输入命令：
npm install -g hexo-cli
hexo -v #查看Hexo版本号

二、本地搭建

Hexo的基本命令

hexo clean #清除生成的博客静态文件
hexo g #生成博客静态文件=命令hexo generate
hexo s #启动博客的本地预览=命令hexo server
hexo d #推送博客到远程仓库=命令hexo deploy
hexo new page xxx #新建名为”xxx”的页面
hexo new xxx #新建名为”xxx”的文章
hexo d -g #生成静态博客并推送到Git远程仓库
hexo s -g #生成静态博客并启动本地预览

1.初始化Hexo
Git Bash输入命令：
hexo init

2.安装依赖
Git Bash输入命令：
npm install

3.本地预览
Git Bash输入命令：
hexo s

打开浏览器，输入http://localhost:4000就可以预览你的博客了（Ctrl+C可退出预览模式）

三、关联Github仓库

1.准备工作
打开GitHub：https://github.com/
登录（没有就注册一个）
新建一个仓库即create repository
新建一个名为xxx.github.io的Public仓库，其中xxx为你的GitHub用户名
勾选项如图所示

2.设置SSH
Git Bash分别输入以下命令：
git config –global user.name “xxx” # xxx是你的Github用户名，注意大小写保持一致
git config –global user.email “your email address” # your email address填写你的Github注册用的邮箱
ssh-keygen -t rsa -C “your email address” # 生成SSH公钥，your email address同上填

三个提示出来后分别按三个Enter键

完成上面步骤后默认生成的密钥在 C:\Users\用户名.ssh\ 目录下，以文本编辑器打开 id_rsa.pub 文件，复制里面的所有内容，后面会用到

在 Github 点开自己头像打开 Settings 页面
选择 SSH and GPG Keys，点击 New SSH Key

Key 中粘贴上面你在本地 id_rsa.pub 文件中复制的所有内容，点击 Add SSH Key

Git Bash输入命令：
ssh -T git@github.com

就会出现，类似图中的提示：(Hi 用户名！You’ve successfully authenticated，but GitHub does not provide shell access.)

3.打开博客目录下的_config.yml （打开方式：记事本）文件，修改最后一行的 deploy 配置信息：

deploy:
type: git
repo:
github: https://github.com/xxx/xxx.github.io.git # Git仓库地址，username是Github用户名
branch: main # 部署到仓库的master(主)分支

4.安装Hexo推送Git插件

Git Bash输入命令：
npm install hexo-deployer-git –save

5.推送到远程仓库

Git Bash分别输入命令：
hexo clean # 清除生成的静态文件
hexo g # 生成新的博客静态文件
hexo s # 启动本地预览
hexo d # 推送博客静态文件到远程仓库

如果每次 hexo d 都要输入 github 账号密码登录，bash 下执行：
git config –global credential.helper store

浏览器地址输入 xxx.github.io 可以访问你的博客了！

四、上传博客
1.下载MarkDownpad2

2.创建一个 .md 文件，并将写好的 .md 文件保存到你的博客文件夹 中的 source文件夹 下的 _posts 中，默认只有一个 hello-world.md；

3.接着在你博客站点文件夹下右键空白处，选择Git Bash Here，输入 hexo g
6 然后 输入 hexo s，完成后 打开浏览器输入 localhost:4000，会发现多了你刚写的那篇博客
7 最后输入hexo d，将本地文件发布到github仓库上

好了，大功告成！
